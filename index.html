Paragraph

A paragraph consists of one or more sentences.

Usage

<p>...</p>

Bold (strong)

Emphasis is the exaggeration of words in a text with a font in a different style from the rest of the text—to emphasize them. It is the equivalent of prosodic stress in speech.

Usage

You can write text in <strong>bold</strong>

Italic/cursive

Italic type is a cursive font normally slant slightly to the right. Italics are a way to emphasise key points in a printed text, or when quoting a speaker a way to show which words they stressed.

Usage

You can write text in <i>italic</i>

Superscript

Superscript is some text that is smaller than their normal line of type and is set slightly above it. Superscripts appear above the baseline.

Usage

<sup>subscript</sup>

Subscript

Superscript is some text that is smaller than their normal line of type and is set slightly below it. Subscripts appear below the baseline.

Usage

<sub>subscript</sub>

Miscellaneous

Badges 42

Badges refer to small, usually numeric callouts next to a navigation item. They can indicate a number of messages or notifications waiting, or the presence of updates. Specifying such badges is as simple as adding a <span> containing the text, with a class of “badge”:

Usage

<span class="badge">42</span>

Link buttons

Link buttons are links styled as buttons.

Usage

<a class="btn btn-default">Click</a>

Examples

btn-default btn-primary btn-link btn-info btn-success btn-warning btn-danger btn-toolbar

Buttons

Link buttons and buttons are styles in the exact same ways.

Usage

<button class="btn btn-default">Click</button>

Examples

Disabled (link-) buttons

Buttons and link buttons can be disabled by adding disables="disabled" to the control. The base color of the button can still be seen on some of the buttons when hovering.

Usage

<a class="btn btn-default" disabled="disabled">Click</a>
<button class="btn btn-default" disabled="disabled">Click</button>
